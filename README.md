# 设备维护系统

## 下载

```git
git clone git@github.com:Lninn/vue-wechat-app.git
```

## 安装

```git
cd /vue-wechat-app
npm install
```

### 调试

```git
npm run serve
```

### 构建

```git
npm run build
```

### 测试

```git
npm run test
```

### Lints检查

```git
npm run lint
```
